About the Project

This project uses ESP32 and relay modules to automatically control classroom lights and fans based on time schedules. It's energy-efficient and affordable.

How It Works

  • RTC or NTP time-based automation
  • ESP32 controls appliances using relay module
  • Optional web dashboard for manual control
